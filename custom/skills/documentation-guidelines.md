# Deprecated: 문서 작성 가이드

이 문서는 더 이상 문서 작성 요청의 진입점으로 사용하지 않는다.

문서 작성, 문서화, docs 작성, ADR, runbook, 아키텍처 문서, 정책 문서, Obsidian 저장 여부 판단은 모두 `docs-writing` 스킬을 먼저 사용한다.

`docs-writing`이 흡수한 내용:

- repo `docs/`와 Obsidian vault의 정본 위치 판단
- 기술 문서, ADR, 운영 정책, runbook, 아키텍처 문서 작성 기준
- 파일명/위치 결정 원칙
- 구조적 글쓰기와 MECE 기본 원칙
- Obsidian 작성이 필요한 경우 `obsidian-writing` 스킬로 위임하는 흐름

사용 규칙:

```txt
문서 작성 요청 → docs-writing → 필요 시 obsidian-writing
```

이 파일은 과거 링크 호환을 위해 남겨둔다. 새 지침은 추가하지 않는다.
